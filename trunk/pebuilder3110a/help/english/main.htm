Main dialog screen


Source path

Drive letter to CD or local/remote path containing the "source" files. The source files must be one of the following versions of Windows:
  • Windows XP Home Edition (must be Service Pack 1 or higher)
  • Windows XP Professional (must be Service Pack 1 or higher)
  • Windows Server 2003, Web Edition
  • Windows Server 2003, Standard Edition
  • Windows Server 2003, Enterprise Edition

If your PC comes with Windows XP preinstalled on the harddisk and your vendor supplies CD/DVD media that will restore the system back into the factory/preinstalled state you may not have a Windows XP installation CD.
However, there is a big change that the Windows installation files are somewhere on your harddisk. Try searching for Windows installation files using the Source/Search menu option.

Custom directory

Select a local or remote path containing any folders and files you want to add/include.
Don't include your windows directory, or any other folder that has file that are in use. Also keep in mind that the files added must fix your target media. For example when creating a bootable CD-Rom you can add something around 300-400MB of extra data.
If you unsure what enter in this fields, then leave it empty!

Output directory

Enter a directory name to "store" the files PE Builder copies for you. Please note that the location your enter is relative to your pebuilder directory.
If you need to specify an absolute path, you must change the "output path absolute" in the Options dialog.

Media output

When you specify "None" the builder will only output the files to your output location and take no further action.
If you want to create an ISO image you must select "Create ISO image" and you must check that a valid filename is entered in the textbox below.
The builder can also burn a CD or DVD media for you if you select "Burn to CD/DVD". If you burn using "cdrecord" you can only burn a CD (not a DVD). Burning to a DVD writer is only supported when burning using starburn!
PE Builder uses mkisofs by Joerg Schilling (GNU-GPL license) for building the ISO image.
PE Builder uses cdrecord by Joerg Schilling (GNU-GPL license) for burning CD.
PE Builder uses the StarBurn SDK library from Rocket Division Software for burning CD/DVD.
