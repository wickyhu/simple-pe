Plugin dialog


This dialog works like this: You select a plugin in the list and use the buttons to process the select item.
Multiple items can be selected in the list, but that is only useful for "Enable/Disable". All the other buttons only work with the first selected item.

Enable/DisableToggles the Enable flag of the (multiple) selected item(s).
ConfigConfigures the plugin, for example with the nu2shell plugin you can use it to enable the boot password protection. The exact funtion of this button can be different for each plugin. It depends on what the plugin creator uses for configuration.
RefreshRe-reads all plugin files. If you have added a plugin manually (not using the Add button) and you don't see the new plugin in the list you can hit the refresh button.
EditOpens the plugin main .inf file in your default text editor. A plugin is based on one (or more) .inf file(s). For example checkout the plugin called Boot Fix (Enabling "Press any key to boot from CD") It is a very small and simple plugin to copy the file bootfix.bin. See Plugin file format for more information about the .inf file format.
AddAdds a new plugin, you must select a cabinet (.cab) file and pebuilder will extract it to the plugin folder. Please check the PE Builder webpage for more plugins and links.
RemoveRemoves a plugin. Warning: this will remove all files and folders for that plugin. For example if you remove the "BartPE Installer v2" plugin, pebuilder will completely remove the plugin\peinst\ folder!
HelpOpens any additional information or help program for the plugin.
