Options dialog

Language Selects the language that pebuider will use. To change the language pebuilder must restart, so if you change the language pebuilder will close.
Mkisofs/CDRecord
Volume ID Enter the volume label which will be added during the ISO build (if left blank, the default label will be "BartPE").
Starburn
Verifying data When enabled this will verify the data written to CD/DVD after burning.
Advanced
Don't build registry When checked, the registry files will not be build.
Don't build files When checked, all files that normally will be copied during building are not copied.
Absolute output path By default the output path is relative to the pebuilder path. When checked, this option allows you to use an absolute path.
Verbose mode This will make the builder engine "talk" more. Use it for troubleshooting the build process. Most of the extra information that is generated is in the english language only, not in the language that you may have selected.
