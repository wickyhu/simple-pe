BartPE Ramdisk

Initialize a ramdisk when you boot up BartPE.

The Ramdisk's size is set to 32 megs and drive letter B: by default. To change this go to the "strings" section from the ramdisk.inf file.
This is an OEM version of the Q-Soft Ramdisk driver, created by Christiaan Ghijselinck.
The driver only runs in BartPE. The maximum size is 96MB. If your system has little memory, for example 64MB, the ramdisk size is automatically set to 4MB. This enables you to run BartPE even on those older notebooks with 64MB of memory.
