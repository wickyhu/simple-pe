How to use the plugin ?
    * 1- Download UltraISO from Here
            http://www.ezbsystems.com/ultraiso/download.htm
    * 2- Setup UtraISO on your system.
    * 3- Uncompress the Plugin cabient.
    * 4- Run the Get_Files.cmd to collect the needed files.
    * 5- Copy this folder to Plugin directory.
    * 6- Start your build and enjoy !

How to clean ?
