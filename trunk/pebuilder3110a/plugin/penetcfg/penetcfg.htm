PE Builder v3 plugin

PE Network Configurator

A network configuration tool for BartPE.
PE Network Configurator (penetcfg.exe) is included with PE Builder with permission from the author.

When using this plugin make sure that you disabled the old network plugin called "Network Support [factory/netcfg/penetcfg]" (plugin\network\network.inf).
