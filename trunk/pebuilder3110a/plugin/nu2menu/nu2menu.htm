PE Builder v3 plugin

Nu2Menu

Nu2Menu is a highly dynamic shell/menu application designed for Windows 2000/XP/PE. It was specially created for PE-Builder. You can also use it as an (autorun) cd menu that uses the same menu file when booting from windows PE. This is the way PE Builder uses it...

All files to use Nu2Menu are included with PE Builder.

Need more info on the available functions in nu2menu? Go here: functions help.

To use a different "GO" button image, just add a file called nu2go.bmp into the plugin\nu2menu directory. PE Builder will automatically copy it when found.
